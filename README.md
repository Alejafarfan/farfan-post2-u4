## Descripción
Este proyecto implementa un formulario de registro con validación del lado del cliente utilizando JavaScript y la Constraint Validation API.

Se valida:

- Nombre completo
- Correo electrónico
- Contraseña
- Confirmación de contraseña
- Teléfono opcional

Además incluye retroalimentación visual y verificación en tiempo real.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- DOM
- Constraint Validation API

## Ejecución del proyecto

1. Clonar el repositorio
2. Abrir la carpeta en Visual Studio Code
3. Ejecutar con Live Server

## Funcionalidades

- Validación en tiempo real con eventos blur
- Validación de formato de correo
- Verificación de contraseña segura
- Confirmación de contraseña
- Mensajes de error personalizados
- Mensaje de registro exitoso
